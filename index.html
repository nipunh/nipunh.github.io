Virtual Painter

Virtual Painter with Hand Gesture is a project based on OpenCV technology that allows users to detect their hands and fingers in real-time. The project leverages the power of the Mediapipe and NumPy libraries to create a virtual canvas where users can use their index finger to draw, and their index and middle fingers to select different colors and an eraser. With Virtual Painter with Hand Gesture, users can immerse themselves in a new and exciting way of drawing, without the need for physical tools like brushes or pencils. The project utilizes advanced image processing techniques to track hand movements and identify specific gestures made by the user. This makes it possible to create a highly responsive and accurate virtual painting experience that mimics the real-world experience of drawing.
